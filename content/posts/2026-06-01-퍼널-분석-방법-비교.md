우리 쇼핑몰 전환율이 계속 1.2% 대에서 맴돌더라. 트래픽은 늘어나는데 매출은 비례해서 안 오르니까 뭔가 구멍이 있다는 생각이 들었다. 그래서 퍼널 분석을 제대로 해보기로 했다.

**결론부터 말하면, 퍼널 분석으로 고객 이탈 지점을 정확히 찾아서 개선했더니 2개월 만에 전환율이 1.2%에서 1.57%로 31% 올랐다.**

## 왜 퍼널 분석을 시작했는지

매월 방문자는 꾸준히 늘어나는데 전환율은 제자리걸음이었다. 구글 애널리틱스만 대충 보고 "어 오늘 매출 좋네" 이런 식으로만 확인하고 있었는데, 이게 문제였던 것 같다.

어디서 고객들이 빠져나가는지 모르니까 개선할 포인트를 찾을 수가 없었다. 그래서 본격적으로 고객 여정을 단계별로 쪼개서 분석해보기로 했다.

## 우리 쇼핑몰 퍼널 분석 과정

**퍼널을 6단계로 나누고 각 단계별 이탈율을 측정했다.**

### 1단계: 퍼널 설계하기

먼저 고객 여정을 다음과 같이 정의했다:
- 랜딩페이지 도착
- 카테고리/상품 목록 조회  
- 상품 상세 페이지 조회
- 장바구니 추가
- 결제 페이지 진입
- 주문 완료

### 2단계: 데이터 수집 도구 셋업

구글 애널리틱스 4 이벤트 추적을 제대로 설정하고, 핫자(Hotjar)도 연결해서 히트맵 데이터도 같이 봤다. 이 과정이 좀 귀찮긴 했는데, 나중에 보니 필수였다.

### 3단계: 1개월간 데이터 수집

충분한 데이터가 쌓일 때까지 기다렸다. 성급하게 며칠 데이터만 보고 판단하면 안 되더라.

## 분석 결과와 충격적인 발견

**가장 큰 이탈 지점이 상품 상세 페이지에서 장바구니 추가 구간이었다. 이탈율이 무려 73%였다.**

전체 퍼널 결과:
- 랜딩 → 카테고리 조회: 82% (이탈율 18%)
- 카테고리 → 상품 상세: 71% (이탈율 29%) 
- 상품 상세 → 장바구니: 27% (이탈율 73%) ← 여기가 문제
- 장바구니 → 결제 진입: 68% (이탈율 32%)
- 결제 → 주문 완료: 91% (이탈율 9%)

핫자 히트맵을 보니까 사람들이 상품 상세 페이지에서 스크롤은 많이 하는데 장바구니 버튼을 잘 안 누르더라. 그리고 페이지를 벗어나려고 할 때 그냥 나가버리는 경우가 대부분이었다.

## 개선 작업 시작

### 문제점 분석
1. 장바구니 버튼이 너무 밑에 있어서 안 보임
2. 상품 옵션 선택이 복잡함  
3. 페이지 이탈 시점에 아무런 인터벤션이 없음

### 해결 방법

**1. 떠다니는 장바구니 버튼 추가**
코드앤버터로 상품 상세 페이지에 하단 고정 바를 만들었다. 가격과 장바구니 버튼이 항상 보이도록 했다. 설정은 진짜 간단했다 - 타겟 페이지를 상품 상세로 지정하고 하단 띠배너 템플릿 선택하고 끝.

**2. 옵션 선택 UI 간소화**
페이지 커스터마이징 기능으로 기존 옵션 선택 부분을 더 직관적으로 바꿨다. 드롭다운을 버튼 형태로 바꾸고 글씨 크기도 키웠다.

**3. 이탈 방지 팝업 설치**
마우스가 페이지 상단으로 올라가면(나가려는 의도) 할인 쿠폰이나 무료배송 혜택을 제안하는 팝업을 띄웠다. 이것도 코드앤버터에서 "페이지 이탈 의도" 트리거로 설정하면 된다.

## 2개월 후 결과

**전환율이 1.2%에서 1.57%로 상승했다. 31% 개선이었다.**

단계별 개선 효과:
- 상품 상세 → 장바구니: 27% → 41% (52% 개선)
- 장바구니 → 결제: 68% → 74% (추가 개선)
- 이탈 방지 팝업 전환율: 약 12%

매출로 환산하면 월 평균 340만원 정도가 늘었다. 솔직히 이 정도 효과가 날 줄은 몰랐다.

## 퍼널 분석하면서 깨달은 것들

**데이터 없이는 추측만 할 뿐이다.** 전에는 "아마 가격이 비싸서 안 사는 것 같다", "상품 사진이 별로인가?" 이런 식으로만 생각했는데, 실제로는 UI 문제였다.

그리고 **작은 개선도 누적되면 큰 효과**가 난다는 걸 실감했다. 버튼 위치 바꾸고 팝업 하나 달았을 뿐인데 이런 결과가 나오다니.

무엇보다 **지속적인 모니터링이 중요**하다. 한 번 고치고 끝이 아니라 계속 데이터를 보면서 새로운 병목 지점을 찾아야 한다.

## 자주 묻는 질문

**퍼널 분석에서 가장 중요한 지표는 뭔가요?**
단계별 이탈율과 전환율입니다. 특히 각 단계에서 몇 %가 이탈하는지 정확히 파악해야 병목 구간을 찾을 수 있어요.

**퍼널 분석 도구 중에 뭘 써야 하나요?**
구글 애널리틱스 4가 기본이고, 더 자세한 분석이 필요하면 핫자나 풀스토리 같은 히트맵 도구를 함께 쓰는 게 좋습니다.

**퍼널 분석 후 개선 효과는 언제쯤 나타나나요?**
보통 2-4주 정도 지나야 의미있는 데이터가 쌓입니다. 급격한 변화보다는 꾸준한 모니터링과 점진적 개선이 중요해요.

## 다음 계획

이제 결제 과정을 더 자세히 분석해볼 예정이다. 여전히 32% 이탈율이 있으니까 더 개선할 여지가 있을 것 같다. 그리고 모바일과 데스크톱 퍼널을 따로 분석해서 디바이스별 최적화도 해보려고 한다.
